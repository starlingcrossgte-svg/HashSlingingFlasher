# HashSlingingFlasher

HashSlingingFlasher is an experimental Android project for vehicle communication, diagnostics, and flashing research.

The current focus is building a simple, reliable communication and testing workflow on Android for Subaru-related adapter and protocol experimentation.

## Current focus

- adapter communication
- protocol research
- logging
- manual command testing
- sequence testing

## Status

This project is in active early development and should be treated as experimental.
