# Provenance

HashSlingingFlasher is an independently developed Android project focused on adapter communication, Subaru protocol research, logging, and command/sequence testing.

## Development approach

- This project is built from original development work in this repository.
- Publicly discussed protocol behavior, packet formats, and adapter communication concepts may be referenced for interoperability and research.
- No proprietary or decompiled third-party source code or assets are intended to be included in this repository.

## Notes

- Similarity at the level of protocol bytes, packet layouts, headers, checksums, timing, or diagnostic sequences can occur because interoperability requires shared protocol behavior.
- Repository history should be used as the source of truth for development progress and implementation provenance.
